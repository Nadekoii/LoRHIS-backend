# LoRHIS - Back-end
## IDE
[IntelliJ]([https://code.visualstudio.com/](https://www.jetbrains.com/idea/))

## System Requirement
Node.js

## Project Setup

```sh
npm init -y                 # Initialize npm project
npm install express @4.17.1 # Install the express package
```

### Compile and Run the Application

```sh
npm start
```
